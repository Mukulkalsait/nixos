
Ctrl+g n        — new window
Ctrl+g x        — kill window
Ctrl+g r        — split right
Ctrl+g d        — split down
Ctrl+g D        — detach session
Ctrl+g c        — enter resize mode → hjkl → Esc to exit
Ctrl+g S        — session menu
Ctrl+g P        — pane menu
Ctrl+g R        — reload config
Ctrl+g <        — built-in pane menu
Ctrl+g >        — built-in window menu
Ctrl+g w        — visual window list
Ctrl+g s        — session switcher
Ctrl+Alt+q      — kill session instantly
Alt+h/l         — prev/next window
Alt+j/k         — swap pane up/down
Alt+p           — floating pane toggle
